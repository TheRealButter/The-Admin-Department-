# Readiness Gates

## Gate 0 — Prototype

The system has:

- offer page
- tracker template
- SOP
- message templates
- report template
- demo dashboard

This is demoable, but not sellable as a working system.

## Gate 1 — Logic-testable

The system has:

- required-field rules
- status rules
- calculated flags
- formula guide
- test records
- pass/fail expectations

This is internally testable, but not yet paid-pilot ready.

## Gate 2 — Live-sheet tested

The system has:

- formulas tested in a live Google Sheet
- dropdowns tested
- conditional formatting tested
- report calculations tested
- test data passes/fails as expected

This is pilot-testable.

## Gate 3 — Private pilot ready

The system has:

- private working environment
- no real data in public repo
- client-approved access rules
- handover tested
- first real/private sample records tested

This can be sold as a controlled paid pilot.

## Current gate status

| System | Current Gate | Meaning |
|---|---|---|
| Invoice Admin | Gate 1 | Logic-testable, not live-sheet tested |
| Sales Admin | Gate 0 | Prototype only |
| Client Admin | Gate 0 | Prototype only |

## Gate rule

No system is sold as working until it reaches Gate 3.
