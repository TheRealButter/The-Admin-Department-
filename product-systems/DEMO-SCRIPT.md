# Demo Script — The Admin Department Systems

Use this script for first sales calls.

## Opening

Most small businesses do not have an admin problem because their people are lazy. They have an admin problem because the work lives in too many places: WhatsApp, Gmail, spreadsheets, documents, and memory.

The Admin Department fixes one department at a time.

## Demo structure

### 1. Show the messy current state

Ask:

- Where do enquiries/invoices/client documents come in?
- Who tracks them?
- What gets missed most often?
- What do you chase manually every week?

### 2. Show the clean system

Pick the relevant department:

- Invoice Admin
- Sales Admin
- Client Admin

Show:

- tracker
- statuses
- next actions
- owner/assignee
- exception or follow-up queue
- message templates
- weekly report

### 3. Explain human approval

Say:

> We do not start by letting automation do everything. We first build a clean workflow your team can trust. Once it works manually, we can add automation behind the scenes.

### 4. Show value

Translate the system into practical outcomes:

- fewer missed follow-ups
- invoices easier to review
- documents filed properly
- clients onboarded consistently
- reports ready without chasing everyone

### 5. Close with audit/setup

Say:

> The next step is an Admin Audit. We map your current workflow, identify the biggest leaks, and recommend the first department to set up.

## Best first close

Start narrow:

> Let us set up one department first. One workflow, one tracker, one report. Once that works, we expand.
