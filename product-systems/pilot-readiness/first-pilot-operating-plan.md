# First Pilot Operating Plan

## Recommended first pilot

Start with **Invoice Admin** unless the prospect has a stronger Sales Admin or Client Admin pain.

## Pilot promise

For one workflow, we will make the admin visible, trackable, and reportable.

## Pilot scope

One system only:

- [ ] Invoice Admin
- [ ] Sales Admin
- [ ] Client Admin

One workflow only:

- one intake process
- one tracker
- one approval process
- one weekly report

## Pilot duration

Recommended: 2 weeks

## Pilot setup steps

### Day 0 — Setup

- confirm pilot system
- copy workbook into private workspace
- configure statuses/owners
- configure access
- add sample/real approved records
- confirm weekly report recipient

### Days 1–3 — Controlled use

- process daily records
- track failed/blocked items
- document confusion points
- confirm approval owners respond

### Days 4–7 — First report

- prepare weekly report
- review stuck records
- fix tracker/template issues
- confirm whether workflow is clearer

### Week 2 — Stabilize

- process more records
- reduce failed/blocked confusion
- test handover again
- prepare pilot closeout report

## Pilot success criteria

- all records have status
- all active records have owner and next action
- risky records do not move forward silently
- weekly report is produced from tracker data
- client understands what is stuck and why
- operator can run system without builder explaining every step

## Pilot failure criteria

- client cannot understand tracker
- records still depend on memory
- weekly report requires guessing
- approval points are unclear
- real data is stored in wrong place
- operator cannot run system after handover

## Pilot closeout options

### Option 1 — Continue managed support

Client keeps using the system with weekly/monthly support.

### Option 2 — Expand to another department

Add Sales Admin or Client Admin after first workflow stabilizes.

### Option 3 — Add light automation

Only after manual workflow is trusted.

### Option 4 — Stop

If workflow does not create enough value, stop without expanding.
