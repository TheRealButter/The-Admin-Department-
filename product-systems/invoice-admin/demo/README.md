# Invoice Admin Demo

This demo package is the first sellable product proof for The Admin Department.

It shows how a messy invoice inbox becomes a clear admin system:

1. supplier documents are captured
2. required fields are checked
3. exceptions are flagged
4. approvals are tracked
5. files are organised
6. a weekly report is produced

## Demo files

- `demo-dashboard.html` — browser demo of the Invoice Admin system.
- `demo-dashboard.css` — styling for the demo.
- `supplier-list.csv` — sample supplier register.
- `exception-queue.csv` — sample missing-info / duplicate / approval issues.
- `filled-weekly-report.md` — realistic filled report for a demo client.
- `demo-walkthrough.md` — 5-minute sales demo script.

## Demo positioning

Do not pitch this as AI.

Pitch it as:

> We set up a simple invoice admin system so supplier documents stop getting lost in email and approvals stop depending on memory.

AI/automation can be added later, but the demo should first prove the workflow.
