# Client Admin System

## Product promise

Onboard clients properly without losing documents, forgetting steps, or chasing the same information manually.

## Best-fit clients

- agencies
- consultants
- accountants/bookkeepers
- service businesses
- legal/admin-heavy firms
- contractors with project onboarding
- property managers onboarding landlords/tenants

## Pain this system fixes

New clients often start through scattered emails, WhatsApp messages, documents, deposits, agreements, folders, and internal notes. The business loses time chasing missing information and clients get an unprofessional onboarding experience.

## Workflow

1. New client is marked as won/accepted.
2. Client Admin tracker record is created.
3. Welcome message is sent.
4. Required documents and information are requested.
5. Folder structure is created.
6. Agreement/deposit/first invoice status is tracked where applicable.
7. Missing items are followed up.
8. Internal handover checklist is completed.
9. Client is marked Active when onboarding is complete.
10. Weekly report shows clients onboarded, stuck, missing info, and next actions.

## Statuses

- New Client
- Welcome Sent
- Documents Requested
- Waiting for Client
- Internal Setup
- Payment/Agreement Pending
- Ready to Start
- Active
- Stuck
- Cancelled

## Human approval rule

A human must approve before:

- sending legal/agreement documents
- confirming payment status
- marking onboarding complete
- accepting incomplete documentation
- escalating difficult client communication

## MVP install components

- onboarding tracker
- document checklist
- folder structure
- welcome templates
- missing-info templates
- weekly onboarding report
- SOP
- demo data

## Automation candidates after manual validation

- onboarding form
- folder creation
- welcome email draft
- missing-info reminders
- checklist reminders
- weekly onboarding summary

## What not to promise

- This does not replace legal review.
- This does not confirm payments unless connected to verified payment data.
- This does not make business decisions for the client.

## Success metrics

- new clients onboarded
- average onboarding time
- missing-document items resolved
- stuck onboarding cases reduced
- first response/welcome time
- internal handover completion rate
