# Sales Admin Demo

This demo package is the second sellable product proof for The Admin Department.

It shows how scattered enquiries become a clear sales admin workflow:

1. enquiries are captured
2. response status is tracked
3. quote needs are visible
4. follow-ups are assigned
5. overdue actions are flagged
6. weekly sales admin reporting is produced

## Demo files

- `demo-dashboard.html` — browser demo of the Sales Admin system.
- `demo-dashboard.css` — styling for the demo.
- `lead-pipeline.csv` — sample lead/enquiry pipeline.
- `follow-up-queue.csv` — sample quote and enquiry follow-up queue.
- `filled-weekly-report.md` — realistic filled report for a demo client.
- `demo-walkthrough.md` — 5-minute sales demo script.
- `pilot-setup-checklist.md` — first paid pilot checklist.

## Demo positioning

Do not pitch this as a CRM or AI sales bot.

Pitch it as:

> We set up a simple sales admin system so enquiries, replies, quotes, and follow-ups stop depending on memory.

Automation can be added later, but the demo should first prove the workflow.
