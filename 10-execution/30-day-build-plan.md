# 30-Day Build Plan

## Week 1 — Company foundation

Build:

- brand positioning
- homepage
- service pages
- Admin Audit offer
- pricing menu
- proposal template
- intake questionnaire
- delivery SOP

Outcome:

The Admin Department is sellable on paper.

## Week 2 — Invoice Admin demo

Build:

- Invoice Admin tracker
- sample supplier database
- document folder structure
- exception queue
- approval status
- weekly report template
- demo using 5 sample invoices

Outcome:

A practical demo you can show bookkeepers, accountants, property managers, and SMEs.

## Week 3 — Sales Admin + Client Admin demos

Build:

- Sales Admin tracker
- enquiry follow-up templates
- quote follow-up queue
- Client Admin onboarding tracker
- welcome email templates
- document checklist
- weekly onboarding report

Outcome:

Three core departments ready for audits and pilots.

## Week 4 — Outreach and pilot

Target:

- 50 bookkeepers/accountants
- 30 property managers
- 30 contractors/service businesses
- 20 consultants/agencies

Goal:

- 20 conversations
- 5 audits/demos
- 1 paid pilot

Pilot rule:

Keep first pilot narrow. One department only. One workflow only. Human approval. Clear report.
