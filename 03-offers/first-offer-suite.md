# First Offer Suite — The Admin Department

## Suite name

Admin Systems Setup

## Suite promise

We install practical admin systems that turn scattered invoices, enquiries, and client onboarding into clear, trackable workflows.

## The three systems

### 1. Invoice Admin Setup

For supplier invoices, receipts, proofs of payment, approvals, filing, and missing-information follow-ups.

Best first pilot when the business has invoice/document chaos.

Includes:

- invoice tracker
- supplier register
- exception queue
- approval workflow
- folder structure
- document naming rules
- weekly invoice report

### 2. Sales Admin Setup

For enquiries, replies, quote tracking, follow-ups, missed-response risks, and simple sales admin reporting.

Best first pilot when the business loses leads or forgets quote follow-ups.

Includes:

- enquiry tracker
- lead statuses
- urgency rules
- quote follow-up queue
- message templates
- weekly sales report

### 3. Client Admin Setup

For new-client onboarding, document collection, agreements, payments, folders, and internal handovers.

Best first pilot when onboarding is messy or clients keep getting stuck before delivery starts.

Includes:

- onboarding tracker
- client status pipeline
- document checklist
- welcome and missing-document templates
- folder structure
- internal handover checklist
- weekly onboarding report

## Best first pilot

Start with one of:

1. Invoice Admin if the client has invoice/document chaos.
2. Sales Admin if the client loses enquiries/follow-ups.
3. Client Admin if the client struggles to onboard clients cleanly.

## Pilot rule

One client. One workflow. One report. Manual first. Automation only after the workflow is trusted.

## Buyer-facing close

> Pick one workflow that is currently messy — invoices, sales enquiries, or client onboarding. We will set up the tracker, statuses, templates, folder/report structure, and handover process so your team can run it properly before we add automation.
